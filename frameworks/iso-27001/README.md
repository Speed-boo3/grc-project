# ISO 27001

## What is it

ISO 27001 is the international standard for information security management. It is published by the International Organization for Standardization. Organisations can get certified by passing an external audit, which proves that their security controls are real and working.

It is the most widely recognised security certification in the world outside the US. If you work as a GRC analyst in Europe, you will almost certainly work with ISO 27001.

## What it covers

ISO 27001 is built around an Information Security Management System (ISMS). This is a set of policies, processes and controls that work together to protect an organisation's information.

The standard includes 93 controls across four themes:

**Organisational controls** cover things like security policies, roles and responsibilities, risk management and supplier relationships.

**People controls** cover screening, training, disciplinary processes and offboarding.

**Physical controls** cover access to buildings, equipment protection, clean desk policies and physical media.

**Technological controls** cover authentication, access rights, encryption, logging, vulnerability management and more.

## The certification process

1. Define the scope of the ISMS
2. Conduct a risk assessment
3. Select and implement controls based on the risks
4. Write a Statement of Applicability listing which controls apply and why
5. Run the ISMS for a period of time
6. Pass a Stage 1 audit (document review)
7. Pass a Stage 2 audit (implementation review)
8. Receive certification, valid for 3 years
9. Annual surveillance audits in years 2 and 3

## Key terms

**ISMS** - Information Security Management System. The overall framework.

**Statement of Applicability (SoA)** - A document listing all 93 controls and stating whether each one applies to your organisation and why.

**Risk register** - A live document tracking all identified risks, their scores and treatment status.

**Control** - A safeguard or countermeasure. Can be technical (a firewall), organisational (a policy) or physical (a lock).
